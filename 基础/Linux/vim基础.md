## 光标

| h          | head |       |
| ---------- | ---- | ----- |
| j          |      |       |
| k          |      |       |
| l          | last |       |
| 30 + h     |      | 移动30次 |
| 30 + Space |      |       |
| 30 + Enter |      |       |
